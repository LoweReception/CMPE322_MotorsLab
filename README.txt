Hello World!
This github repo contains the source files for the Motors Lab from CMPE 322 (Briggs, F2014).

The project was build on the chipkit MX4Pro (Digilent) which uses the pic32f512L microprocessor.

The following is used to intialize on-board peripherals:
PERIPHERAL INITALIZATION TEMPLATE
   set OSCx	-> set open-drain control
   set PORTx	-> set bits in PORTx SFR
   set TRISx	-> set tri-state control
